# ED_Lista01.03
## Fatorial Recursiva

Este desafio consiste em criar uma aplicação, em Java, que exiba o resultado do fatorial de um número.

```
Requisitos do projeto:

1. Arquitetura MVC;
2. O cálculo do fatorial deve ser realizado por uma função recursiva;
3. O valor de entrada deve ser um número natural menor ou igual a 12 (devido as limitações da memória por empilhar muitas funções);
4. Declarar, via comentário, a condição de parada da função.
```
